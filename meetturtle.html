turtle.goto(x,y)

Tells the turtle to go to the given x coordinate and y coordinate

turtle.position()

Gives you back the x and y of the turtle’s location

        Example:

                x,y = turtle.position()

                # x will have the value of the x coordinate of the turtle

                # y will have the value of the y coordinate of the turtle

                

                a,b = turtle.position()

                # a will have the value of the a coordinate of the turtle

                # b will have the value of the b coordinate of the turtle

turtle.pendown()

It puts the pen of the turtle down so when the turtle moves it draws a line

turtle.penup()

It puts the pen of the turtle up so when the turtle moves it doesn’t draw a line

turtle.hideturtle()

It hides the turtle shape

turtle.showsturtle()

It shows the turtle shape

turtle.speed(speed)

Set the turtle’s speed to an integer value in the range 0..10. If no argument is given, return current speed.

  • speed  is an integer in the range 0..10 or a speedstring (see below)

Set the turtle’s speed to an integer value in the range 0..10. If no argument is given, return current speed.

If input is a number greater than 10 or smaller than 0.5, speed is set to 0. Speedstrings are mapped to speedvalues as follows:

  • “fastest”: 0
  • “fast”: 10
  • “normal”: 6
  • “slow”: 3
  • “slowest”: 1

Speeds from 1 to 10 enforce increasingly faster animation of line drawing and turtle turning.

Attention: speed = 0 means that no animation takes place. forward/back makes turtle jump and likewise left/right make the turtle turn instantly.

Mouse Actions

turtle.onclick(fun, btn=1, add=True)        

onclick is a function that will run fun every time you click on the turtle itself, Where:

  • fun is the name of the function you want to run every time you click on the turtle itself the fun should have only two arguments/inputs x and y
  • btn is number of the mouse button, for left button the value should be 1
  • add most of the time set it to True which means add new fun to run, so you can run more than one fun at the same time

Example:

                turtle.onclick(turtle.goto,btn=1,add=True)

turtle.onscreenclick(fun, btn=1, add=True)

onscreenclick is a function that will run fun every time you click anywhere on the screen, Where:

  • fun is the name of the function you want to run every time you click anywhere on the screen the fun should have only two arguments/inputs x and y
  • btn is number of the mouse button, for left button the value should be 1
  • add most of the time set it to True which means add new fun to run, so you can run more than one fun at the same time

        

Example:

                turtle.onscreenclick(turtle.goto,btn=1,add=True)

turtle.ondrag(fun, btn=1, add=True)

  • fun is the name of the function you want to run every time you start dragging the turtle itself the fun should have only two arguments/inputs x and y
  • btn is number of the mouse button, for left button the value should be 1
  • add most of the time set it to True which means add new fun to run, so you can run more than one fun at the same time

        

        Example:

                turtle.ondrag(turtle.goto)

Keyboard Actions

In order to make these functions work add this line turtle.getscreen().listen() at the end before turtle.mainloop()

        Example:

                so your code at the end of the file should have something similar to this

                turtle.getscreen().listen()

                turtle.mainloop()

turtle.getscreen().onkeypress(fun, key)

onkeypress is a function that will run fun every time you press the key key on the keyboard, where:

  • fun – is the name of the function you want to run every time you press the key key on the keyboard, fun should have no arguments/inputs
  • key – a string the letter or the name of the key: key (e.g. “a”) or key-symbol (e.g. “space”)

        Example:

                def myfun():

                        turtle.penup()

                turtle.getscreen().onkeypress(myfun,”Up”)

                turtle.getscreen().listen()

sadek.pendown() #run pendown for sadek the turtle
